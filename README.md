SuperGPT (Astra) is an intelligent chatbot system built using Flask, BeautifulSoup, and Groq API, designed to assist students and users by answering queries about KIIT University and related topics.
It combines web scraping, vector-based search (TF-IDF + cosine similarity), and AI-powered conversation to give accurate, context-aware responses.
Steps: 
1. Download the zip file.
2. Paste your API KEY(groq api)
3. Scrap the website you want
4. Now interact with subject specifit chatbot.
